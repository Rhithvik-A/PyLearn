‚ö° Start Coding with Python

What is Python?

Python is a beginner-friendly, powerful programming language used to build apps, games, websites, automation tools, and AI systems. It's one of the most popular languages in the world!

üìù Python Syntax

Python syntax is clean and easy to read. Unlike other languages, Python doesn't use curly braces or semicolons. Instead, it relies on indentation to define code blocks. 

print("Python is easy!")

üí° Tip: Code runs from top to bottom. Indentation (spaces) tells Python which lines belong together.

üéØ Your First Program

Let's write the classic "Hello, World!" program: 

print("Hello, World!")

üéâ Congratulations! If this runs successfully, you're officially a Python programmer!

‚å®Ô∏è Input & Output

Python can interact with users through input and output: 

name = input("Enter your name: ")
print("Hello,", name)

üí° input() - Gets data from the user

üí° print() - Displays output to the screen



üí¨ Comments

Comments are notes in your code that Python ignores. They help you and other developers understand what the code does.

Single-Line Comments

Use the # symbol to create a single-line comment: 

# This is a comment
age = 14  # This stores the user's age
print(age)

Multi-Line Comments

For longer explanations, use triple quotes (""" or '''): 

"""This is a multi-line comment.
It can span multiple lines.
Useful for longer explanations."""
print("Comments don't affect output")

üí° Best Practice: Write clear comments to explain why you're doing something, not just what you're doing.



üì¶ Variables

Variables are containers that store data values. Think of them as labeled boxes where you can keep information to use later in your program.

Creating Variables

age = 14
name = "Alex"
is_student = True
print(age, name, is_student)

üìã Variable Naming Rules

‚úì

Must start with a letter or underscore (_)

‚úì

Can contain letters, numbers, and underscores (A-z, 0-9, _)

‚úì

Are case-sensitive (age, Age, and AGE are different)

‚úó

Cannot start with a number

‚úó

Cannot use Python keywords (like if, for, while)

Displaying Variables

Use print() to display variable values: 

x = 14
print(x)    # Output: 14

‚ö° Multiple Assignment

Assign the same value to multiple variables: 

x = y = z = 100
print(x, y, z)

Assign multiple values to multiple variables: 

name, age, is_student = "Alex", 14, True
print(name, age, is_student)

üí° Best Practice: Use descriptive variable names like user_age instead of x to make your code more readable.

Data types define what kind of data a variable holds.

Python has many in-built data types:

  1. Text Type - str

  2. Numeric Types - int, float, complex

  3. Boolean Type - bool

  4. Sequence Type - list, tuple, range

  5. Mapping Type - dict

  6. Set Type - set, frozenset

  7. None Type - NoneType

Numbers

Python has three numeric types:

  • Integers - int

    üí° Integers are whole numbers without a decimal point. Example: x = 14

  • Floats - float

    üí° Floats are numbers with a decimal point. Example: x = 14.5

  • Complex - complex

    üí° Complex numbers are numbers with a real and imaginary part. Example: x = 5j

Slicing

Example:

üí° The first number is the index of the first character, and the second number is the index of the last character.

üí° If you omit the second number, the slice will go to the end of the string.

‚úÖ Booleans

Booleans represent one of two values: True or False. They're essential for making decisions in your code.

Comparison Operations

When you compare values, Python returns a Boolean result: 

print(10 > 9)     # True
print(10 == 9)    # False
print(10 < 9)     # False

Using Booleans in Conditions

Booleans are commonly used in if-else statements: 

a = 200
b = 33

if b > a:
    print("b is greater than a")
else:
    print("b is not greater than a")

The bool() Function

Convert any value to a Boolean using bool(): 

print(bool("Hello"))    # True
print(bool(15))         # True
print(bool(0))          # False
print(bool(""))         # False

üìä Truthy vs Falsy Values

‚úì Truthy (True)

  • ‚Ä¢ Non-empty strings
  • ‚Ä¢ Numbers (except 0)
  • ‚Ä¢ Non-empty lists, tuples, sets
  • ‚Ä¢ Non-empty dictionaries

‚úó Falsy (False)

  • ‚Ä¢ Empty strings ("")
  • ‚Ä¢ Zero (0)
  • ‚Ä¢ Empty collections ([], {})
  • ‚Ä¢ None

üí° Remember: Almost everything with content is True, and empty or zero values are False.

üìã Lists

Lists are one of Python's most versatile data structures. They store multiple items in a single variable and allow you to organize, access, and manipulate collections of data efficiently.

Creating a List

fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # apple
print(fruits[1])    # banana
print(fruits[2])    # cherry

‚ú® Key Features of Lists

‚úì

Ordered

Items have a defined order with index positions

‚úì

Mutable

You can add, remove, or modify items

‚úì

Flexible Types

Can store any data type (int, float, string, etc)

‚úì

Allows Duplicates

Same value can appear multiple times

üìè List Length

Use the len() function to find how many items are in a list: 

fruits = ["apple", "banana", "cherry"]
print(len(fruits))    # Output: 3

üîç Access List Items

Access individual items in a list using their index number. Remember, Python uses zero-based indexing - the first item is at index 0. 

fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # apple (first item)
print(fruits[1])    # banana (second item)
print(fruits[2])    # cherry (third item)

üí° Remember: List indexing starts at 0, not 1!

Negative Indexing

Use negative indices to access items from the end of the list. -1 refers to the last item: 

fruits = ["apple", "banana", "cherry"]
print(fruits[-1])    # cherry (last item)
print(fruits[-2])    # banana (second from end)
print(fruits[-3])    # apple (third from end)

Slicing Lists

Extract a range of items using the slice operator [start:end]: 

fruits = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(fruits[2:5])    # ['cherry', 'orange', 'kiwi']
print(fruits[:4])     # ['apple', 'banana', 'cherry', 'orange']
print(fruits[2:])     # ['cherry', 'orange', 'kiwi', 'melon', 'mango']

‚ö†Ô∏è Note: The end index is exclusive - fruits[2:5] includes indices 2, 3, and 4, but not 5.

Negative Slicing

You can also use negative indices in slices: 

fruits = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(fruits[-4:-1])    # ['orange', 'kiwi', 'melon']

Check if Item Exists

Use the in keyword to check if an item is in a list: 

fruits = ["apple", "banana", "cherry"]
print("banana" in fruits)      # True
print("grape" in fruits)       # False

‚úèÔ∏è Change List Items

Lists are mutable, meaning you can modify their contents after creation. Change a specific item by referring to its index: 

fruits = ["apple", "banana", "cherry"]
fruits[1] = "orange"
print(fruits)    # ['apple', 'orange', 'cherry']

Change a Range of Values

Replace multiple items at once using slice notation: 

fruits = ["apple", "banana", "cherry", "lemon"]
fruits[1:3] = ["orange", "kiwi"]
print(fruits)    # ['apple', 'orange', 'kiwi', 'lemon']

Flexible Replacement

The number of new items doesn't have to match the number being replaced: 

# Replace 1 item with 3 items (list grows)
fruits = ["apple", "banana", "cherry"]
fruits[1:2] = ["orange", "kiwi", "melon"]
print(fruits)    # ['apple', 'orange', 'kiwi', 'melon', 'cherry']

                            
                        
# Replace 2 items with 1 item (list shrinks)
fruits = ["apple", "banana", "cherry"]
fruits[1:3] = ["orange"]
print(fruits)    # ['apple', 'orange']

Insert at Position

Use insert() to add an item at a specific position without replacing: 

fruits = ["apple", "banana", "cherry"]
fruits.insert(1, "orange")
print(fruits)    # ['apple', 'orange', 'banana', 'cherry']

üí° Tip: insert() shifts existing items to the right, while slice assignment replaces items.

‚ûï Add List Items

Python provides several methods to add items to a list. Choose the right one based on your needs.

append() - Add to End

Use append() to add a single item to the end of the list: 

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits)    # ['apple', 'banana', 'cherry', 'orange']

insert() - Add at Position

Use insert() to add an item at a specific index: 

fruits = ["apple", "banana", "cherry"]
fruits.insert(1, "orange")    # Insert at index 1
print(fruits)    # ['apple', 'orange', 'banana', 'cherry']

extend() - Add Multiple Items

Use extend() to add all elements from another iterable: 

fruits = ["apple", "banana", "cherry"]
more_fruits = ["orange", "kiwi"]
fruits.extend(more_fruits)
print(fruits)    # ['apple', 'banana', 'cherry', 'orange', 'kiwi']

üí° Quick Reference: Use append() for one item, insert() for specific position, extend() for multiple items.

‚ûñ Remove List Items

Python offers multiple ways to remove items from a list. Each method serves a different purpose.

remove() - Remove by Value

Removes the first occurrence of the specified value: 

fruits = ["apple", "banana", "cherry"]
fruits.remove("banana")
print(fruits)    # ['apple', 'cherry']

If there are duplicates, only the first one is removed: 

fruits = ["apple", "banana", "cherry", "banana"]
fruits.remove("banana")
print(fruits)    # ['apple', 'cherry', 'banana']

pop() - Remove by Index

Removes and returns the item at the specified index (or the last item if no index is given): 

fruits = ["apple", "banana", "cherry"]
removed = fruits.pop(1)    # Remove item at index 1
print(fruits)              # ['apple', 'cherry']
print(removed)             # banana

del - Delete by Index or Slice

Use the del keyword to remove items by index: 

fruits = ["apple", "banana", "cherry"]
del fruits[1]
print(fruits)    # ['apple', 'cherry']

You can also delete the entire list: 

fruits = ["apple", "banana", "cherry"]
del fruits
# fruits no longer exists!

clear() - Remove All Items

Empties the list but keeps the list object: 

fruits = ["apple", "banana", "cherry"]
fruits.clear()
print(fruits)    # []

üí° Quick Reference: Use remove() for value, pop() for index with return, del for index, clear() to empty list.

üîÑ Sort Lists

Python provides powerful methods to sort and organize list items in various ways.

Alphabetical/Numerical Sorting

The sort() method sorts the list in ascending order by default: 

fruits = ["orange", "mango", "kiwi", "pineapple", "banana"]
fruits.sort()
print(fruits)    # ['banana', 'kiwi', 'mango', 'orange', 'pineapple']

Reverse Order

Sort in descending order using the reverse=True parameter: 

fruits = ["orange", "mango", "kiwi", "pineapple", "banana"]
fruits.sort(reverse=True)
print(fruits)    # ['pineapple', 'orange', 'mango', 'kiwi', 'banana']

Or use reverse() to reverse the current order: 

fruits = ["orange", "mango", "kiwi"]
fruits.reverse()
print(fruits)    # ['kiwi', 'mango', 'orange']

Case-Insensitive Sorting

By default, sort() is case-sensitive. Use key=str.lower for case-insensitive sorting: 

fruits = ["banana", "Orange", "Kiwi", "cherry"]
fruits.sort(key=str.lower)
print(fruits)    # ['banana', 'cherry', 'Kiwi', 'Orange']

üí° Tip: sort() modifies the list in place. Use sorted() to create a new sorted list without changing the original.

üìã Copy Lists

Creating a copy of a list is important when you want to modify one without affecting the other. Python offers several methods:

Method 1: copy()

original = ["apple", "banana", "cherry"]
copy_list = original.copy()
print(copy_list)    # ['apple', 'banana', 'cherry']

Method 2: list()

original = ["apple", "banana", "cherry"]
copy_list = list(original)
print(copy_list)    # ['apple', 'banana', 'cherry']

Method 3: Slice [:]

original = ["apple", "banana", "cherry"]
copy_list = original[:]
print(copy_list)    # ['apple', 'banana', 'cherry']

üîó Join Lists

Combine multiple lists into one using different methods:

Using the + Operator

The simplest way to concatenate lists: 

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]
combined = list1 + list2
print(combined)    # ['a', 'b', 'c', 1, 2, 3]

Using extend()

Add all elements from one list to another: 

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]
list1.extend(list2)
print(list1)    # ['a', 'b', 'c', 1, 2, 3]

üìö List Methods Reference

Here's a complete reference of all built-in list methods in Python:

Method Description
append() Adds an element at the end of the list
clear() Removes all elements from the list
copy() Returns a shallow copy of the list
count() Returns the number of elements with the specified value
extend() Adds elements from an iterable to the end of the list
index() Returns the index of the first element with the specified value
insert() Adds an element at the specified position
pop() Removes and returns the element at the specified position

üéâ Congratulations! You now know all the essential list operations in Python. Lists are one of the most powerful and frequently used data structures. Practice using them to become proficient!



üì¶ Tuples

Tuples are one of Python's built-in data types used to store multiple items in a single variable. They are similar to lists, but the key difference is that tuples are immutable.

Creating a Tuple

fruits = ("apple", "banana", "cherry")
print(fruits[0])   # apple
print(fruits[1])   # banana
print(fruits[2])   # cherry

‚ú® Key Features of Tuples

Ordered

Items have a fixed order

Immutable

Items cannot be changed after creation

Allows Duplicates

Same value can appear multiple times

Any Data Type

Can store mixed data types

üìè Tuple Length

fruits = ("apple", "banana", "cherry")
print(len(fruits))   # 3

üîç Access Tuple Items

Tuples use zero-based indexing, just like lists. 

fruits = ("apple", "banana", "cherry")
print(fruits[0])
print(fruits[1])
print(fruits[2])

Negative Indexing

fruits = ("apple", "banana", "cherry")
print(fruits[-1])
print(fruits[-2])
print(fruits[-3])

Slicing Tuples

fruits = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(fruits[2:5])
print(fruits[:4])
print(fruits[2:])

‚ö†Ô∏è Note: End index is excluded in slicing.

Negative Slicing

fruits = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(fruits[-4:-1])

Check if Item Exists

fruits = ("apple", "banana", "cherry")
print("banana" in fruits)
print("grape" in fruits)

üö´ Tuples Cannot Be Changed

fruits = ("apple", "banana", "cherry")
fruits[1] = "orange"   # ERROR

üîÅ Modify Tuple (Workaround)

fruits = ("apple", "banana", "cherry")
temp = list(fruits)
temp.append("orange")
fruits = tuple(temp)
print(fruits)

üîÑ Sort Tuples

fruits = ("orange", "mango", "kiwi", "banana")
sorted_fruits = sorted(fruits)
print(sorted_fruits)

üìã Copy Tuples

original = ("apple", "banana", "cherry")
copy_tuple = tuple(original)
print(copy_tuple)

üîó Join Tuples

t1 = ("a", "b", "c")
t2 = (1, 2, 3)
joined = t1 + t2
print(joined)

üìö Tuple Methods Reference

Method Description
count() Returns number of occurrences
index() Returns index of value

üîÄ if, elif & else

Conditional statements are used to make decisions in Python. They allow your program to execute different blocks of code based on conditions.

üîπ if Statement

The if statement executes a block of code only if the condition is true. 

age = 18

    print("You are eligible to vote")

üîπ if else Statement

    print("Eligible to vote")

    print("Not eligible to vote")

üîπ if elif else Statement

Use elif (else if) to check multiple conditions.

    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 60:
    print("Grade C")
else:
    print("Fail")

üí° Rule: Python checks conditions from top to bottom and stops at the first true one.

üîÑ Loops

Loops allow you to repeat code automatically, saving time and reducing redundancy. They're essential for processing collections and automating repetitive tasks.

For Loop

Use a for loop when you know exactly how many times you want to repeat: 

for i in range(1, 6):
    print("Count:", i)

# Output:
# Count: 1
# Count: 2
# Count: 3
# Count: 4
# Count: 5

You can also loop through lists, strings, and other collections: 

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

While Loop

Use a while loop when you want to repeat until a condition becomes false: 

count = 1
while count <= 5:
    print(count)
    count += 1

# Output: 1 2 3 4 5

‚ö†Ô∏è Warning: Make sure your while loop has a way to end, or it will run forever (infinite loop)!

Loop Control Statements

break

Exits the loop immediately:

for i in range(10):
    if i == 5:
        break
    print(i)  # Prints 0 to 4

continue

Skips the current iteration:

for i in range(5):
    if i == 2:
        continue
    print(i)  # Prints 0, 1, 3, 4


‚öôÔ∏è Functions

Functions are reusable blocks of code that perform specific tasks. They help you organize code, avoid repetition, and make your programs more maintainable.

Creating a Function

Use the def keyword to define a function: 

def greet(name):
    print("Hello,", name)

# Calling the function
greet("Alex")    # Output: Hello, Alex
greet("Sam")     # Output: Hello, Sam

Function Parameters

Functions can accept multiple parameters: 

def add_numbers(a, b):
    result = a + b
    print("Sum:", result)

add_numbers(5, 3)    # Output: Sum: 8

Return Values

Functions can return values using the return statement: 

def multiply(a, b):
    return a * b

result = multiply(4, 5)
print(result)    # Output: 20

Default Parameters

You can set default values for parameters: 

def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alex")              # Output: Hello, Alex!
greet("Sam", "Hi")         # Output: Hi, Sam!

üí° Best Practice: Use functions to break down complex problems into smaller, manageable pieces. This makes your code easier to read, test, and debug.
